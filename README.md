1. What is the difference between var, let, and const?

Answer:

var: Function-scoped, hoisted, can be reassigned and redeclared.

let: Block-scoped, hoisted (temporal dead zone), can be reassigned but cannot be redeclared in the same scope.

const: Block-scoped, hoisted (temporal dead zone), cannot be reassigned or redeclared. Objects/arrays can be mutated.










2. What is the spread operator (...)?

Answer:

The spread operator ... allows you to expand arrays, objects, or iterables into individual elements.

 









 3. What is the difference between map(), filter(), and forEach()?

 Answer:

map(): Loops over an array, returns a new array with modified elements.

filter(): Loops over an array, returns a new array with elements that pass a condition.

forEach(): Loops over an array, does not return anything, used for side-effects.










4. What is an arrow function?

Answer:

An arrow function is a shorter syntax for writing functions in JavaScript. It also does not have its own this.










5. What are template literals?

Answer:

Template literals are strings wrapped in backticks `. They allow string interpolation and multiline strings.
